# 一．项目定位

本项目面向电商平台售后服务和用户反馈分析场景，构建一个支持用户咨询、商家处理和平台管理的智能客服系统。用户端支持订单问题咨询、退换货申请、售后进度查看和评价提交；商家客服端支持查看用户问题、AI辅助回复、工单处理、售后分类和服务记录；管理员端支持商品知识库、售后规则、客服账号、工单统计和评价分析管理。系统通过大模型识别用户问题意图，如物流延迟、退货退款、商品质量、发票问题等，并结合商品说明、售后政策和FAQ知识库生成参考回复；同时对用户评价进行情感分析和主题归类，帮助商家发现高频问题。

# 二．终端形态与用户场景

| **终端** | **载体**             | **用户**    | **核心场景**               |
| ------ | ------------------ | --------- | ---------------------- |
| 用户端    | 微信小程序 / H5 / App内嵌 | 消费者、买家    | 订单咨询、退换货申请、售后进度查询、评价提交 |
| 商家客服端  | Web管理后台（PC）        | 客服人员、售后专员 | 问题回复、工单处理、售后分类、服务记录    |
